Using Javascript to Make a Request

JavaScript

Ajax is what we will be using to take our API data and insert it into our html.

Lets see what a basic request will look like:

                            
        var request = new XMLHttpRequest();
        var ID = "Yq3v6n";
        var key = "?key=57ac3d20257053737a469c6962fd60be";

        request.open(GET,"http://api.brewerydb.com/v2/beer/"+ ID + key, true )

        request.AddEventListener("load", function{
            var information = JSON.parse(request.responsetext);
            console.log(information.overview);
        });

There is not too much going on here, but it might still be a little confusing so I will go through everything

The Vars

These are all of the variables that you I am using to make this code work, the most important one is var request = new XMLHttpRequest();. This is an object that makes it possible to send http requests. The other variables are just to make it easier for me to organize my code.

Open and addEventListener

request.open(GET,"http://api.brewerydb.com/v2/beer/"+ ID + key, true ) is where we actually send our request to the server. The next part request.AddEventListener("load", function{ is telling the computer to wait until the page is loaded before it does the next section of code which stores the information given and then displays it in the console

Right click on the page and look at the console, you should be able to find all the information gathered by the request.
