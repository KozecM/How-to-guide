Step 2: Making a Request

The Request

When you want some information. You need to search for it here is an example: http://api.brewerydb.com/v2/beer/Yq3v6n?key=mykey. Lets break down what is happening:

  • api.brewery.com/v2/: This part tells you the api too get information from.
  • beer/: this part tells you what type of request you are making. It can be many different things like a search or a guild
  • Yq3v6n: This part is the reference to a specific item.
  • ?key=Yq3v6n?key=mykey: You replace mykey with the key that you got earlier, it telles the api that you are regstered and are allowed to access the info.
